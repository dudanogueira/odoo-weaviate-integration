========================
Product Weaviate Search
========================

.. |badge1| image:: https://img.shields.io/badge/maturity-Beta-yellow.png
    :target: https://odoo-community.org/page/development-status
    :alt: Beta

    :alt: OCA/product-attribute

|badge1| |badge2| |badge3|

Syncs Odoo products to a `Weaviate <https://weaviate.io>`_ vector database and
replaces the default backend and eCommerce product search with Weaviate hybrid
(BM25 + vector) search powered by OpenAI embeddings.
